# Itotori App

The Itotori app is the Studio surface for localization runs and their artifacts.
It owns catalog identity, DB-backed project/import/locale-branch workflows,
deterministic QA and protected-span checks, patch-ready export, runtime evidence
ingestion, glossary/context lookup, benchmark and cost reporting, and typed API
schema validation.

The live localization path uses OpenRouter through TanStack. Its current
screens are **Play**, **Wiki**, **Results**, **Refine**, and **Settings**.
