# Kaifuu Encrypted Matrix Fixture

This fixture is synthetic, deterministic, and public-safe. It contains tiny
archive-like byte strings that exercise Kaifuu encrypted-input readiness
diagnostics without retail game bytes, extracted scripts, screenshots, audio,
fonts, video, helper dumps, private paths, or private keys.

All archive bytes, key labels, helper outputs, and profiles are generated by
`fixtures/generate-kaifuu-encrypted-public-fixtures.mjs`. The public fixture
keys are fixture-only labels for deterministic CI and are safe to commit because
they unlock only these generated public byte strings.

Siglus expected outputs in `expected/siglus-*.json` are regenerated by this
generator from the current `kaifuu-cli` detector/profile/inventory/parser
commands. Run `node fixtures/generate-kaifuu-encrypted-public-fixtures.mjs
--check` to fail on stale committed Siglus JSON instead of preserving old bytes.

Covered detector families:

- KiriKiri/XP3-like plain, encrypted, compressed, and unknown archive signals.
- Siglus-like `Scene.pck` plus `Gameexe.dat` key-profile signals.
- RPG Maker MV/MZ encrypted asset suffixes plus `System.json` encryption flags.
- Wolf RPG Editor-like protected archive signals.
- BGI/Ethornell-like BURIKO container and encrypted marker signals.
- Unknown archive-like aggregate signals and marker-only negative detector cases.

The helper-result fixtures conform to the KAIFUU-085 helper result schema. The
negative key-profile fixtures intentionally fail validation and exist only to
prove redacted diagnostics for raw-key-looking and private-path-looking refs.
